# README

## Deploy to Heroku

- 'heroku login'
- 'git init'
- 'heroku create journoethics --stack cedar --region us --buildpack https://github.com/AdmitHub/meteor-buildpack-horse.git'
- 'heroku git:remote -a journoethics'
- 'git add '
- 'git add .'
- 'git commit -m "First Deploy"'
- 'git push heroku master'
